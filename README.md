# 🚀 Xeon RAM Configurator (V2/V3/V4)

Профессиональный интерактивный калькулятор и симулятор BIOS для тонкой настройки систем на базе Intel Xeon (LGA2011 / 2011-3). Инструмент позволяет рассчитать идеальные тайминги для стабильной работы и максимальной пропускной способности памяти.

## ✨ Основные возможности

### 🛠 Модульная архитектура (Системные вкладки)

Проект спроектирован как полноценная утилита для настройки платформы:

- **RAM Tool:** (Текущий модуль) Полная настройка таймингов и вольтажа.
- **Power Manager:** Управление лимитами энергопотребления (TDP/EDP).
- **Fan Configuration:** Настройка кривых оборотов кулера.
- **IIO / PCI Configuration:** Управление линиями PCI-Express и прерываниями.

### 🧠 Умный RAM Engine

Автоматический расчет параметров на основе конфигурации железа:

- **Поддержка поколений:** Intel Xeon V2 (DDR3), V3 и V4 (DDR4).
- **Динамический расчет частот:** от 1333 MHz до 2400 MHz.
- **RFC Logic:** Уникальный алгоритм расчета `tRFC` в зависимости от объема планок и форм-фактора материнской платы (**ATX** vs **mATX**).
- **Пресеты:** Безопасный, Оптимальный (Balanced), Агрессивный и Экстремальный (Ultra).
- **Custom Mode:** Режим ручного ввода для оверклокеров с мгновенным пересчетом ПСП (Bandwidth).

### 🖥 Визуализация BIOS

Аутентичный интерфейс в стиле классических BIOS (Ami/Phoenix) отображает:

- **Первичные тайминги:** CL, tRCD, tRP, tRAS.
- **Вторичные тайминги:** tWR, tRFC, tRRD, tRTP, tWTR, tFAW, tRC, tCWL.
- **Служебные параметры:** Voltage, Command Timing (1N/2N), Refresh Rate (tREFI).
- **Метрики:** Реальная частота, Режим канальности (QUAD/DUAL), Пропускная способность (GB/s).
